Wordle
A simple recreation of the popular word puzzle game Wordle, where players guess a hidden 5-letter word in six attempts.

How to Play
You have 6 attempts to guess the secret 5-letter word.
After each guess:
Green = correct letter in the correct position
Yellow = correct letter in the wrong position
Grey = letter not in the word

Features
Random word selection from a word list
Input validation (only real 5-letter words allowed)
Colour-coded feedback system
Simple and clean user interface
Keyboard support (if applicable)
